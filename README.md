# 🎬 Movie Explorer

A modern **Movie Explorer web app** that allows users to browse movies based on their preferences.
Users can explore trending movies, filter by genre, and view detailed movie information such as poster, rating, and overview.

This project uses **HTML, CSS, and Vanilla JavaScript** and fetches movie data from the **TMDB API**.

---

## 🚀 Features

* 🎥 Browse **Trending Movies**
* 🎭 Filter movies by **Genre**
* ⭐ View **Movie Ratings**
* 📝 Read **Movie Overview**
* 🔄 **Hero slider** that changes movies automatically
* 📱 **Responsive card layout**
* ⚡ Fast API-based movie loading

---

## 🛠️ Tech Stack

* **HTML5**
* **CSS3**
* **JavaScript (Vanilla JS)**
* **TMDB API**
